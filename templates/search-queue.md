# Search Queue

Only searches that do one of these stay in the active queue:
- establish the paper's originality and literature position
- support a central empirical claim
- prevent overclaiming in a vulnerable section

Everything else waits until a draft reveals a real gap.

## Status Key
- `[ ]` — Not started
- `[-]` — Searched but no good result yet (note what was tried)
- `[~]` — Found candidate(s), not yet downloaded/read
- `[x]` — Done — paper downloaded, note created
